Con lo visto anteriormente en la sección de selectores, la primer regla afecta a todos los elementos <p> que tengan el id "normal". A este le aplica la familia de la fuente, el tamaño y le aplica negrita. Esto lo vemos en la primera línea de la página.

Luego encontramos dos elementos que tienen el id "destacado" un table y un p. Al utilizarse el selector universal, esto nos dice que sera aplicado a todos los elementos con ese id. En este caso le aplica un borde solido, de color azul y con un ancho de 2px.

Y por último, vemos que la tercera regla tiene #, por lo que afecta a los elementos con el determinado id. En este caso es el último párrafo, al que se le pone un fondo celeste y las letras de color rojo. Vemos que el hecho de poner el * delante del id en el selector no afecta al funcionamiento, ya que se aplica igualmente a todos los elementos con ese id.
